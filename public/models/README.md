# 3D Models Directory

Place your `laptop.glb` model file here.

## Where to Get a Laptop Model

### Free Options:
1. **Sketchfab** - https://sketchfab.com/3d-models?features=downloadable&sort_by=-likeCount&q=laptop
2. **CGTrader Free** - https://www.cgtrader.com/free-3d-models/laptop
3. **TurboSquid Free** - https://www.turbosquid.com/Search/3D-Models/free/laptop

### Creating Your Own:
1. Open Blender
2. Model a simple laptop
3. File > Export > glTF 2.0 (.glb)
4. Save as `laptop.glb` in this directory

## Model Specifications

- Format: GLB (glTF Binary)
- Recommended file size: < 2MB
- Triangle count: < 50,000
- Centered at origin (0, 0, 0)

## Temporary Workaround

The app will display a placeholder cube if the model is not found. Replace it with your actual model when ready.
